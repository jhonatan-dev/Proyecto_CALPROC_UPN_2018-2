# Descripción
- Proyecto de curso Calidad de Procesos de Software en UPN 2018-2
- Hecho con Java

# Módulos
* Registrar Préstamo
* Registrar Pago de Préstamo

# Requerimientos
* MySQL versión 5.0 o superior como motor de Base de datos.
* Eclipse versión 2018.12 o superior como IDE.
* Spring Boot versión 2.0 o superior.
